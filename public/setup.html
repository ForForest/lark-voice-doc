1 火山引擎语音 未配置

让 Beeni 能「听」你说话、用语音「说」回你（ASR + TTS）。
⚠️ 最容易卡的一步：在同一个应用上，要分别「开通服务」——『流式语音识别(大模型)』和『大模型语音合成 Seed-TTS』两个都要开。只拿到 key 但没开通，会鉴权失败。
去火山语音控制台拿 App ID / Token ↗

2 豆包方舟 (Ark) 未配置

智商最高的那颗大脑——负责跟你聊创意 + 把讨论画成画板。
⚠️ 拿到 key 后，还要去「开通管理」开通这两个模型：doubao-seed-2-0-pro（对话）和 doubao-seed-2-0-mini（画板）。没开通模型，调用会 NotFound。
创建 API Key ↗ 开通模型 ↗

3 飞书画板 可选

把讨论实时落到飞书白板。这步不是填 key，是在终端跑一次浏览器授权。不弄也能语音聊，只是画板落不进飞书。
npx -y @larksuite/cli@latest config init --new
npx -y @larksuite/cli@latest auth login --domain docs,markdown,wiki
飞书开放平台 ↗ 跑完上面两条命令、浏览器点授权即可。
火山 + 豆包 都填好就能开始。
